# Sistema de Pedidos NFC — Black Burguer

Solução digital integrada para automação de atendimento por aproximação, gerenciamento de estoque em tempo real e controle de produção de pedidos voltada a estabelecimentos comerciais do setor alimentício.

## 📊 Visão Geral do Projeto
O projeto consiste em uma plataforma desenvolvida de forma incremental ao longo dos semestres acadêmicos do curso de Análise e Desenvolvimento de Sistemas da Faculdade de Americana (FAM). [cite_start]O sistema foi validado em ambiente real utilizando o restaurante piloto "Black Burguer", localizado em Santo André/SP[cite: 78, 115, 116].

A arquitetura resolve três dores centrais do negócio:
1. [cite_start]**Atendimento:** Substituição de comandas analógicas de papel por cartões inteligentes NFC[cite: 50, 87, 89].
2. [cite_start]**Logística:** Controle automatizado de SKUs e insumos perecíveis[cite: 120, 157, 181]. [cite_start]Durante a validação piloto do módulo de estoque, o proprietário registrou uma **redução de 40% no tempo gasto com a auditoria de inventário** (de 4h para 2,5h de contagem)[cite: 129, 141, 177].
3. [cite_start]**Produção:** Gerenciamento reativo de pedidos com comunicação direta entre a linha de frente e a cozinha[cite: 199, 206, 238].

## ⚙️ Diferenciais e Tecnologias Consolidadas
* [cite_start]**Comunicação e Hardware:** Protocolo NFC por indução magnética, emulação de tags físicas NTAG e fluxo local de escrita/leitura em curto alcance[cite: 68, 85, 93].
* [cite_start]**Persistência e Arquitetura:** Banco de dados relacional MySQL com mapeamento de integridade referencial[cite: 122, 171, 301].
* [cite_start]**Motores de Negócio:** Aplicações de lógica estruturada e orientação a objetos desenvolvidas em Python[cite: 122, 171, 213].
