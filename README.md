# Sistema de Pedidos NFC — Black Burguer

Solução digital integrada para automação de atendimento por aproximação, gerenciamento de estoque em tempo real e controle de produção de pedidos voltada a estabelecimentos comerciais do setor alimentício.

## 📊 Visão Geral do Projeto
O projeto consiste em uma plataforma desenvolvida de forma incremental ao longo dos semestres acadêmicos do curso de Análise e Desenvolvimento de Sistemas da Faculdade de Americana (FAM). O sistema foi validado em ambiente real utilizando o restaurante piloto "Black Burguer", localizado em Santo André/SP.

A architectural resolve três dores centrais do negócio:
1. **Atendimento:** Substituição de comandas analógicas de papel por cartões inteligentes NFC.
2. **Logística:** Controle automatizado de SKUs e insumos perecíveis. Durante a validação piloto do módulo de estoque, o proprietário registrou uma **redução de 40% no tempo gasto com a auditoria de inventário** (de 4h para 2,5h de contagem).
3. **Produção:** Gerenciamento reativo de pedidos com comunicação direta entre a linha de frente e a cozinha.

## ⚙️ Diferenciais e Tecnologias Consolidadas
* **Comunicação e Hardware:** Protocolo NFC por indução magnética, emulação de tags físicas NTAG e fluxo local de escrita/leitura em curto alcance.
* **Persistência e Arquitetura:** Banco de dados relacional MySQL com mapeamento de integridade referencial.
* **Motores de Negócio:** Aplicações de lógica estruturada e orientação a objetos desenvolvidas em Python.
